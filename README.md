# AdminPortalProject

Installation
git clone https://github.com/JLopezz3/AdminPortalProject
cd admin-portal
npm i
npm start

Branching
git branch {branch-name}
git checkout {branch-name}
git push --set-upstream origin {branch-name}

Workflow
git checkout {branch-name}
git add .
git commit -m {message}
git push

Pulling from master
git checkout {branch-name}
git pull origin master
